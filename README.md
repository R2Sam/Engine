# Engine

This 2D engine or perhaps more aptly framework is based off raylib.

## Features

- Scenes and scene manager
- Ecs
- Event system
- Resource manager
- Basic animations
- Basic networking using enet
- Lua scripting

A basic example (though somewhat dated) using some of these features can be found at https://github.com/R2Sam/Snake.
Better in progress documentation can be found at https://engine.speediskey.eu/annotated.html
